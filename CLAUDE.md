# 🚀 CLAUDE CODE - FULL STACK BOOSTED CONFIG

## 🛠 Tech Stack Focus
- **Frontend:** React, Next.js, TypeScript, Tailwind CSS.
- **Backend:** Node.js, Express, NestJS, Python (FastAPI/Django).
- **Architecture:** Microservices, Serverless, REST, GraphQL.
- **Database:** PostgreSQL, MySQL, MongoDB, Redis.
- **DevOps:** Docker, Kubernetes, CI/CD (GitHub Actions), AWS/Vercel.

## 🎯 Coding Standards & Logic
- **Clean Code:** Applique les principes SOLID, DRY et KISS.
- **TypeScript:** Typage strict exigé. Évite l'usage de `any`. Préfère les interfaces aux types pour les objets.
- **Performance:** Optimise le rendu (memoization) et les requêtes (indexation, limitation des jointures lourdes).
- **Security:** Sanitize les entrées, utilise les variables d'environnement, prévient les injections SQL et XSS.
- **Components:** Préfère les composants fonctionnels et les Hooks personnalisés.

## 🧠 Workflow Optimization Rules
1. **Sequential Thinking:** Décompose systématiquement les problèmes complexes en sous-tâches logiques avant de coder.
2. **Context Awareness:** Analyse toujours les fichiers existants avant de proposer une nouvelle implémentation pour éviter les doublons.
3. **Validation:** Propose ou lance des tests unitaires (Jest/Vitest) après chaque modification majeure.
4. **Documentation:** Ajoute des commentaires JSDoc pour les fonctions complexes et maintient le fichier README à jour.

## ⚡ Assistant Behavior
- **Concise:** Pas de bavardage inutile. Priorité au code et aux explications techniques directes.
- **Proactif:** Si une modification de fichier peut en casser un autre, préviens immédiatement.
- **Local First:** Privilégie les solutions et outils qui tournent localement sans dépendances externes payantes.
